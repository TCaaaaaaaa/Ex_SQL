# 摘要

SQL（Structured Query Language）是大数据与数据库课程的核心教学内容，但传统在线评测系统（Online Judge）多基于结果集比对，缺乏对代码逻辑的语义级诊断，且无法根据学生的认知状态提供个性化学习路径。针对上述问题，本文提出了一种基于可拓学理论的 SQL 错误诊断与自适应学习推荐方法，并设计实现了智能辅助教学系统 ExtenicSQL。

首先，本文引入可拓学基元理论，构建了 SQL 语句的**物元模型（Matter-element Model）**，将非结构化的 SQL 代码映射为包含特征元（如选择列、数据源、约束条件）的结构化形式。在此基础上，利用**可拓变换（Extension Transformation）**理论建立差异分析模型，通过增、删、扩、缩等变换算子，精准识别用户语句与标准答案的语义偏差，并生成具有解释性的纠错建议。

其次，针对学习路径缺乏个性化的问题，本文提出基于**多维关联函数（Multidimensional Correlation Function）**的认知状态评估模型。综合考量答题正确率、时间效率及记忆衰减因素，计算知识点的关联度，并利用**可拓集合**理论将学生的认知状态动态划分为稳定正域、可拓域（临界域）和负域。系统依据域划分结果，结合知识图谱技术，自适应推荐复习或进阶题目，实现“因材施教”。

最后，基于 Vue 3 和 FastAPI 框架开发了系统原型。实验结果表明，该系统能够有效识别 SQL 语义错误，并通过可视化知识图谱直观反馈学习状态，显著提升了学生的学习效率与知识掌握度，验证了可拓学在代码语义分析与智能教育领域的应用价值。

**关键词**：SQL 教学；可拓学；错误诊断；个性化推荐；知识图谱

# Abstract

SQL (Structured Query Language) is a core component of big data and database curricula. However, traditional Online Judge (OJ) systems primarily rely on result set comparison, lacking semantic-level diagnosis of code logic and failing to provide personalized learning paths based on students' cognitive states. To address these issues, this thesis proposes an SQL error diagnosis and adaptive learning recommendation method based on **Extenics** theory, and designs an intelligent auxiliary teaching system named **ExtenicSQL**.

Firstly, by introducing the primitive theory of Extenics, this paper constructs a **Matter-element Model** for SQL statements, transforming unstructured SQL code into a structured form containing characteristic elements (e.g., selection columns, data sources, constraints). On this basis, a difference analysis model is established using **Extension Transformation** theory. By applying transformation operators such as addition, deletion, expansion, and contraction, the system accurately identifies semantic deviations between user statements and standard answers, generating interpretable error correction suggestions.

Secondly, addressing the lack of personalization in learning paths, this paper proposes a cognitive state assessment model based on **Multi-dimensional Correlation Functions**. By integrating factors such as accuracy, time efficiency, and memory decay, the system calculates the correlation degree of knowledge points. Using **Extension Set** theory, it dynamically categorizes students' cognitive states into stable positive fields, extension fields (critical fields), and negative fields. Based on this domain partition and knowledge graph technology, the system adaptively recommends review or advanced exercises, achieving "individualized instruction."

Finally, a system prototype is developed using Vue 3 and FastAPI frameworks. Experimental results demonstrate that the system effectively identifies SQL semantic errors and intuitively reflects learning states through a visualized knowledge graph. It significantly improves students' learning efficiency and knowledge mastery, verifying the application value of Extenics in the fields of code semantic analysis and intelligent education.

**Keywords:** SQL Education; Extenics; Error Diagnosis; Personalized Recommendation; Knowledge Graph
